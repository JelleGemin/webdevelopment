    Opdracht 3

  1. Waarom staat de tekst in de blokken horizontaal gecentreerd?:

    2. Door de text-align: center. De tekst wordt hierdoor gecentreerd.

  2. Waarom heeft vertical-align: center geen effect?:

    3. Vertical-align heeft enkel effect op inline-block & inline elementen. Div is een block element.

  3. Wat doet vertical-align:middle?

    4. Vertical-align: middle zorgt ervoor dat een inline-block of inline element in het midden van de pagina wordt geplaatst.

  4. Simpele manier om tekst te centreren: text-align:center;
  5. Welke truct bestaat er om meerdere regels te centreren?

    6. Aan de elementen een display: table-cell toekennen. Daarna een vertical-align: center. Dit is te zien bij de div's bovenaan de pagina.

  6. Waarom krijd de regel voor class 'tall' voorrang?

    7. CSS = Cascading Style Sheets. Dit wilt zeggen dat het css bestand wordt gelezen van boven naar beneden. Hierdoor kan je makkelijk elementen overschrijven. Aangezien dat div element eerst een height krijgt van 100px en daarna een height van 150px, wordt het element 150px hoog.
